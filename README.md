# 图片转PDF工具

一个功能强大的纯前端图片转PDF工具，支持无限数量图片上传，智能顺序检测，专业PDF生成。

## ✨ 功能特点

### 🚀 核心功能
- **无限制上传**: 支持无数量限制的图片上传
- **智能顺序检测**: 自动检测文件名编号顺序
- **封面特殊处理**: 智能识别封面文件（cover、封面等）
- **专业PDF生成**: 支持多种页面尺寸和自定义设置
- **实时预览**: 拖拽排序，实时预览效果
- **✨ 图片清晰度增强**: 4级可调的图片锐化和清晰度提升
- **📝 自定义文件名**: 智能文件名建议和实时验证
- **🎯 简洁输出**: 生成的PDF无页码，更加简洁美观

### 📱 界面特色
- **现代化设计**: 美观的渐变背景和毛玻璃效果
- **响应式布局**: 完美适配手机、平板、电脑
- **流畅动画**: 丰富的CSS动画和交互效果
- **直观操作**: 拖拽上传、可视化排序

### ⚙️ PDF设置
- **页面尺寸**: A4、A3、A5、Letter、Legal、自定义
- **页边距设置**: 0-50mm可调
- **图片质量**: 60%-90%压缩质量选择
- **页面方向**: 自动检测、纵向、横向
- **✨ 清晰度增强**: 关闭/轻度/中度/强度四级增强
- **📝 自定义文件名**: 支持中英文，智能建议和验证
- **预设模式**: 漫画、文档、照片、网页模式

## 🎯 使用说明

### 1. 上传图片
- 点击上传区域选择图片
- 或直接拖拽图片到上传区域
- 支持 JPG、PNG、GIF、WEBP 格式

### 2. 顺序检测
- 工具会自动检测文件名中的数字顺序
- 自动识别封面文件（如cover.jpg）
- 如果检测不到顺序，会提示用户选择处理方式

### 3. 预览排序
- 在预览区域查看所有图片
- 拖拽图片调整顺序
- 使用排序按钮快速排序

### 4. 设置PDF参数
- 选择页面尺寸或自定义尺寸
- 设置页边距和图片质量
- 选择页面方向
- **✨ 选择图片清晰度增强级别**
- **📝 输入自定义PDF文件名**
- 使用预设模式快速配置

### 5. 图片增强预览
- 选择清晰度增强级别后可查看效果对比
- 实时预览原图与增强后的效果
- 轻度增强适合一般图片，强度增强适合模糊图片

### 6. 文件名设置
- 输入自定义PDF文件名（支持中英文）
- 系统提供智能文件名建议
- 实时验证文件名合法性
- 自动生成最终文件名预览

### 7. 生成PDF
- 点击"生成PDF"按钮
- 查看进度条显示生成进度（含图片增强进度）
- 自动下载生成的PDF文件（无页码，简洁美观）

## 🔧 特殊功能说明

### 智能顺序检测
- 检测文件名中的数字编号（001, 02, 3等）

- 特殊处理封面文件：
